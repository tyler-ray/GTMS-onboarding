Step 2:

What do we require?

GTMS requires all the basic information in 210 and 214 EDI files.

Download GTMS 210 Mapping File

GTMS requires something special:

  • For 210: N9*CR*XX -> Customer identification number (Given by GTMS), is placed in the XX
  • For 214: L11*XX*11 -> Customer identification number (Given by GTMS), the same as for the 210, goes in place of the XX

Step 3:

Where to Drop off and Pick up EDI?

Our FTP File system is more than simple. Drop your EDI files and you're done!

  1. We'll need to whitelist your IP. Send your IP, and company name to:
    1. Tyler.Ray@gtms
  2. First choose what type of file you're dropping. 210? or 214? Then drop into the Inbound folder.
    • Think of this like GTMS's inbox.
  3. See your 997 form in the 997 file. Think of this like a receipt.
    • Dropped files will move to Archive for the past 3 months for reference
